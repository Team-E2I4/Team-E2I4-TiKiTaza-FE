# 제목을 아랫줄에 작성, 제목 끝에 마침표 금지, 무엇을 했는지 명확하게 작성
# 본문 작성시 한줄 밑에 -본문 내용 작성

################
# [✨feat]: 새로운 기능 추가
# [🐛fix]: 버그 수정
# [📝docs]: 문서 수정
# [💡comment] : 주석 추가, 수정, 삭제
# [♻️refactor]: 코드 리팩토링 (기능에 영향을 주지 않을 때)
# [🎨style]: css 스타일링
# [🔨chore]: 빌드 부분 혹은 패키지 매니저 수정사항
# [🗑️remove]: 코드 또는 파일 제거할 때
# [🚚change]: 파일 혹은 폴더 구조, 이름을 변경할 때
################
